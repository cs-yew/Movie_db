# mov.db
Web Programming Assignment (PHP,mySQL)

Mov.db is an online film/movie database.

<h2>Notes</h2>
<ul>
  <li>Database is located at sql folder</li>
  <li>video(.mp4/.mpeg) can be added into video folder</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Login</li>
  <li>Admin & User permission</li>
  <li>Admin - Edit/Delete movies details</li>
  <li>Add film details</li>
  <li>User - search</li>
  <li>Search engine</li>
  <li>Cool Output design</li>
  <li>Lists of films page</li>  
</ul>

<h2>Softwares used</h2>
<ul>
  <li>Any Web server - WAMP Server/Xamp</li>
  <li>Any Text editor-Sublime text/Notepad++/Visual studio</li>
</ul>
